# 文法本の構造（LLM構造化プロンプトの前提知識）

対象の本: **HSK公認テキスト４級**（中国語文法の教科書、説明は日本語）。単語本（新HSK1~4級単語トレーニングブック）は `word-book-structure.md` を参照。OCR結果を LLM で vocab-trainer 用 draft JSON に構造化する際の判断基準。

## 全体構造

- 本は **4つの Part** で構成される。
- 各 Part には複数の **Point** がある。
- Point N の近くには **Pointの題名**（例:「中国語の基本的な語順」）がある。
  - 該当ページの一番上にあるとは限らないが、**文字サイズが大きい**。
  - **この題名で group を作る**（draft の `groups` に使う）。
  - 題名の下の小さな文字の説明文は無視してよい。
- 各ページの左端・右端に上から「Part 1 / Part 2 / Part 3 / Part 4」と印字されているが、**無視する**。

## 注目する「4点セット」

Point 内に複数登場する。上から順に 1 → 2 → 3 → 4 の縦並び:

1. **文法要素の説明** — 「介詞＋名詞＋述語」「从＋場所/時間＋動作を表す述語」のように、日本語の文法用語・「＋」表記・中国語文法表現が混ざる行。
   - 要素間が横に離れているため、OCRでは**複数ボックスに分かれることが多い**（例:「从」「＋」「場所/時間」「＋」「動作を表す述語」が別ボックス）。ボックスの**高さ（y座標）が近ければ同一文**とみなす。
2. **ピンイン** — 3の文章のピンイン。単語間の距離で複数ボックスに分かれうる（例:「Hòuhuǐ」「yě」「láibují」「le.」）。必ずしも単語ごとに分かれるとも限らない。
3. **中国語文章** — 1の文法要素の例文。同様に分割されうる（例:「后悔」「也来不及了。」）。
4. **日本語翻訳** — 3の日本語訳。

→ 1 が draft の `statement`、3+4 が `examples[]`（sentence / translation）、2 は捨てる（ピンインはクラウド側 smart-add が生成）。

`statement` の文法要素は印字どおりではなく**正規化する**:

- 具体的な中国語の機能語はそのまま（从, 别, 把, 了 …）
- 文法的役割は英語の**小文字**短縮表記。固定マッピングは `server/llm.py` の `ELEMENT_ABBREVIATIONS` が正（主語→s, 動詞/述語→v, 目的語→o, 名詞→n, 形容詞→adj, 副詞→adv, 動詞フレーズ→vp, 量詞→m, 数詞→num）。マッピング外の役割も同スタイルの小文字略語にする
- 意味的なスロットは簡潔な日本語名詞（人物, 場所, 時間, 抽象的な目標 …）
- 冗長な説明句は短縮する: 「動作を表す述語」→ v、「名詞（場所）」→ 場所。例: 「从＋場所/時間＋動作を表す述語」→ `从+場所/時間+v`
- 区切りは半角 `+`。プロンプト指示に加えて、LLM出力後に `normalize_statement()` がコード側でも変換を保証する

`description` は**箇条書き**（1行1ポイント、各行 `- ` 始まり。意味・使用条件・ニュアンス・注意点を別々の項目に）で簡潔にする。ただし本が使う**比喩・日本語での例え・言い換えは漏らさず**残す。

**抜け漏れ防止**: 「＋」を含む行はほぼ確実に1（文法要素の説明）であり、4点セットの起点。全行を再走査して「＋」行が候補化されているか確認する。

**3点セット**: たまに1（文法要素の説明）が無く、「ピンイン・中国語文章・日本語訳」の3点だけの場合がある。周囲の説明文を読んで判断し、既出の文法要素と**別の要素**を示すなら独立した候補として `statement` を自作（同じ正規化記法）、同じ要素なら該当候補の `examples` に追加する。

### 4点セットの下に来るもの（ケースバイケース）

- **追加説明**（日本語）: 例「也は「〜も（同じように)…」という意味だけでなく、「〜ても…」という意味で動詞が２つある文にも使えます。」
  - 追加説明の中に中国語文章が登場することがある。この場合、日本語翻訳は中国語文の**下ではなく右側**に来ることが多い。
- **単語注**: 例「*网：ネットワーク（インターネット）」
- **補助説明**: 「（介詞フレーズ）」「結果補語」など、1の文法要素を3の例文に対応付ける注記。

→ 同じ文法要素に関する情報なら活用し（description に反映など）、別の文法要素なら区別する。

## セッションと項目（無視する）

- 1つの Point には複数の**セッション**、1つのセッションには複数の**項目**があり、項目の中に上の4点セットが複数登場する。
- セッションには「文全体の述語を見つけよう!」「注意点」などの見出し文が入り、その下に項目（項目の下にも説明文が付くことが多い）や長い説明文（改行で別ボックスになりうる）が続く。
- **セッション見出し・項目見出し自体は draft に使わない**（適切に無視する）。

## 表

- 多くの場合 **4列**:
  1. 空白 or 「級」
  2. **単語** — 中国語。単語の上にピンインが乗ることもある。行が折り返す場合あり。
  3. **意味** — 日本語の意味。
  4. **例文・補足説明** — 中国語文章。上にピンイン、下に日本語訳（1つのボックスに収まるとは限らない）。その下にたまに単語説明や注意点が日本語で入る。
- OCR結果だけでは表と分かりにくい点に注意。
